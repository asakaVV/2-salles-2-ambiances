# 2-salles-2-ambiances

## Pour créer et lancer l'exécutable

Se mettre à la racine du projet et faire les commandes suivante :

```
mkdir build
cd build
cmake ..
make
cd ../bin
./DSDA
```

## Manuel utilisateur

```
Z -> avancer
Q -> aller à gauche
S -> reculer
D -> aller à droite

maintenir clique droit et bouger la souris pour tourner la caméra

A -> pour quitter
```
